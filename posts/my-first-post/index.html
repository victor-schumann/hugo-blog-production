hugo-demo-development commands

basic hugo commands

hugo new site quickstart
cd quickstart
git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
echo "theme = 'ananke'" >> hugo.toml
hugo server
hugo new content content/posts/my-first-post.md

config and deploy

baseURL = "https://victor-schumann.github.io/blog-demo-production/"
git submodule add -b main git@github.com:victor-schumann/blog-demo-production.git public
git add .
git commit -m "message"
git push
hugo

Now go to github and publish!
